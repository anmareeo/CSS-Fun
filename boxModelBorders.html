Pink - To more easily show the content, padding and border, I have styled the p element to have a background color of blue on all three boxes. The content is what is between the p element tags. The padding is shown by the original color of the box/the name of the box. I have now added a border with this syntax- border: 10px ridge. The first is the size of the border, the second is the style. Please note that if no color is selected after the size, the default color will be grey. There is no default style, so if you try to create a border with only a size and a color, you will see nothing. A border style must be selected in order for anything to show up.

Turquois - I have now added a border with this syntax- border: 5px blue dotted. The first is the size of the border, the second is the color, and the third is the style. I also used border-radius to round the corners a bit. The higher the px, the more rounded the corners.

Just as with the padding, we can select different border sizes, colors and styles on each side. The syntax- border-width: 2px 20px 5px 10px. border-color: gold purple turquoise red; border-style: solid groove double dashed; Again it starts at the top and goes clockwise. Note that I did decrease the padding here to more easily show the border styles and colors

Here I have used the syntax that includes two parameters where the first defines the top and bottom and the second defines the left and right. I have also used a really high radius number and made the height and width the same to turn the box into a circle. I also decreased the font size so the text would fit better.
