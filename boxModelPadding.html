Yellow - This padding was set using another type of syntax where the first number is for the top and bottom and the second number is for the left and right. padding: 100px 10px. Also, note that the height of the yellow box was coded at 50 px and the height of the turquoise box was coded at 100px. The reason the yellow box is taller is because of the padding. The padding will add to the height and width of the div/box and the height and width of the content/text. In closing, note that each box/color shows a different way to add padding to meet your needs. Just remember that if you want different padding on each side, the order starts at the top and goes clockwise....top, right, bottom, left.
